Instal·lar laravel

	mkidr laravel
	cd laravel
	git init
	git remote add origin https://github.com/laravel/laravel.git
	git pull origin master / git fetch origin


Instal·lar composer:

	> curl -sS https://getcomposer.org/installer | php
	> mv composer.phar /usr/local/bin/composer
	> composer install


Genera Key Encriptació (config/app.php):

	> php artisan key:generate


Generar el fitxer que crea les taules (app/database/time_create_users_table.php):

	> php artisan migrate:make create_users_table


Generar les taules

	> php artisan migrate


Desfer la creacio de les taules

	> php artisan migrate:rollback
